Documentation Of CSS HPCODERS

CSS add's style's to your HTML elements that allow's you to manipulate there placement.

There are 3 Main Ways to target HTML elements.

    Targeting Tag

  1. Im going to give a Header tag the styles of color:red; and background of gray;

    	
		CSS

		h1 {
			color: red;
			background-color: blue; /*This color is selected using Hexidecimal numbers.   ff  ff  ff   or RR GG YY. As you change the values the amount of Red Green or Yellow they change color*/
		}
	
	
    This h1 tag has a color of red and background of blue!

    Find your favorite color!

    You can do this with any tag. But this will target all instances that the tag is in the HTML as long as it does not have a class or id.
    Let me target all paragraphs and make there color green and font-size!

    	
		CSS
		p {
			font-size:24px; // Font-size is mainly used with (px) or pixels
			color:green;
		}
	
	
    This Paragraph text is green with a font of 24 pixels

    Classes and ID's

      Classes (.)

    1. You can apply a class to multiple HTML elements. 
      2. 		HTML
			<p class="className"> Text here </p>
		CSS
		.className{
			color:red;
			font-size:30px;
			background-color:gray;
		}

      Text Here

      ID's (#)

    2. ID's can only be used once on a webpage. Use these cautiously for something like the logo. 
      3. 		HTML
			<p id="idName"> Text here </p>
		CSS
		#idName{
			color:green;
			font-size:16px;
			background-color:orange;
		}

      Text Here
