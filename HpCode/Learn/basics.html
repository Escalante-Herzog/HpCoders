(===) checks that both the value and type of the expression your checking is true.
Basically it checks to see two numbers are stored the same. Where 1 is an interget 1.0 is a floting number. Meaning that they are not equal using (===) but equal (==)

Difference Between (==) vs (===) in JavaScript


	0 == false   // true
	0 === false  // false, because they are of a different type
	1 == "1"     // true, automatic type conversion for value only
	1 === "1"    // false, because they are of a different type
	null == undefined // true
	null === undefined // false
	'0' == false // true
	'0' === false // false
